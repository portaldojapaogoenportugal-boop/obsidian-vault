# 「つもりだ」と「予定だ」の違い

## 本質的な対立軸

本質的な違いは確定度ではなく、**内的（主観的決意）か、外的（客観的手配）か**。

| | **つもりだ** | **予定だ** |
|---|---|---|
| **性質** | 主観的・内的 | 客観的・外的 |
| **確定の種類** | 話し手の内的決意として固まっている | 外部的に手配・スケジュール化されている |
| **意志の有無** | 意志を直接含む | 意志を直接表さない |

> 「予定だ」はその予定が自分の意志に基づくものか否かは別にして、客観的な事実として述べるだけ。
> ――国際交流基金「日本語教育通信」

### 「と思っている」を加えた三段階

| 表現 | 性質 |
|---|---|
| 〜**と思っている** | 漠然とした意向（まだ固まっていない） |
| 〜**つもりだ** | 内的に固まった意志・決意 |
| 〜**予定だ** | 外的に確定したスケジュール |

> **注意**：「つもりだった」（過去形）は未実現の含意が出るが、これは現在形「つもりだ」の確定度の問題ではなく別の用法。

## ハンドブックの記述（初級日本語ハンドブック p.138）

**「Vル/ナイ＋予定だ」**：他の人と相談の上決めたことや公的な決定事項を表す。

> 今度の週末にクラス全員でお花見に行く [×つもりだ／○予定だ]

**Cf.** 「Vル/ナイ＋つもりだ」は話し手の個人的な心づもりを表す。

### この記述についての注記

「他の人と相談の上」という説明は上記の例文（クラス全員でのお花見）には当てはまるが、一般原則としてはやや過剰に限定的。

> 「明日は映画を見る**予定だ**」（個人の計画にも予定だは使える）

本質は「相談したかどうか」より**「スケジュールとして客観化されているか」**（*objetivado como agenda / plano concreto*）という点にある。

## キーワード（PT-PT）

| 日本語 | PT-PT |
|---|---|
| 予定 | *agenda / plano agendado* |
| スケジュール・客観的な事実 | *agenda / facto objetivo* |
| つもり | *intenção / determinação* |
| 意図・内的な意志 | *intenção / vontade interior* |
| 主観的・内的 | *subjetivo / interior* |
| 客観的・外的 | *objetivo / exterior* |
| 内的決意として固まっている | *resolução interna já consolidada* |
| 外部的に手配・スケジュール化されている | *organizado e agendado externamente* |
| スケジュールとして客観化されている | *objetivado como agenda / plano concreto* |
| 漠然とした意向 | *intenção vaga / difusa* |
| 意志を直接含む | *expressa diretamente a vontade* |
| 意志を直接表さない | *não expressa diretamente a vontade* |

## 日本人は意識して使い分けているか

**ほぼ無意識。**「予定」「つもり」という語の意味がすでに以下の感覚を内包しているため、論理的に判断するのではなく語感から自然に使い分けが生まれる。

- 予定 → スケジュール・客観的な事実（*agenda / facto objetivo*）
- つもり → 意図・内的な意志（*intenção / vontade interior*）

---

## 注記：国際交流基金の表現について

国際交流基金は「つもりだ」が「客観的意味合いを併せ持つ」と述べているが、これは絶対的な意味での客観性ではなく、「したい」のような純粋な欲求表現と比較した**相対的な表現**と解釈するのが適切。「つもりだ」の本質は主観的な内的決意であり、「したい」より固い、という意味で客観性に近い側面があるにすぎない。
