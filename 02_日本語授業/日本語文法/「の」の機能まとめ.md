# 「の」の機能まとめ

日本語の「の」は複数の異なる機能を持つ。統語的には共通点があっても、意味機能で区別するのが正確。以下、ポルトガル語（PT-PT）対訳つきで整理する。

---

## 1. 助詞「の」— 連体修飾（の genitivo / atributivo）

`名詞1 ＋ の ＋ 名詞2` の形で、名詞どうしを結びつける。PT-PT の **de** に対応することが多い。

| 用法 | 例 | PT-PT |
|---|---|---|
| 所有 | 私**の**車 | o carro **de** mim → o **meu** carro |
| 所属・性質 | 日本語**の**先生 | professor **de** japonês |
| 同格（＝） | 友人**の**山田さん | o Yamada, meu amigo |
| 主格の代用（連体修飾節の中） | 私**の**書いた本 ＝ 私**が**書いた本 | o livro que **eu** escrevi |

> [!note] 主格の の
> 連体修飾節の内部では、主語を示す「が」を「の」で置き換えられる。
> 背**の**高い人 ＝ 背**が**高い人 / a pessoa que é alta。

---

## 2. 準体助詞「の」— 体言の代用（nominalizador / pró-forma）

連体形（動詞・形容詞の連体形、または `な`）を受けて、そのまま名詞句をつくる。統語的な仕組みは①と別で、「の」自身が名詞相当になる。**指示対象がモノ寄りかコト寄りか**で下位区分される。

### 2a. 代用の「の」— pronominalização / elipse do nome

省略された**名詞（もの・人）**の代わりに「の」が立つ。「〜のやつ／〜な方」の感覚。

| 品詞 | 接続 | 例 | PT-PT |
|---|---|---|---|
| イ形容詞 | 連体形＋の | 安い**の** | o (mais) **barato** |
| ナ形容詞 | な＋の | きれいな**の** | o/a **bonito(a)** |
| 動詞 | 連体形＋の | 買った**の** | o que **comprei** |
| 指示 | 連体詞＋の | 赤い**の** | o **vermelho** |

- 置換テスト：安いの → 安い**もの**（○）／安い**こと**（×）
- PT-PT の対応：**名詞が消えて定冠詞 o/a が残る**（o [casaco] mais barato）。
- 例：安い**の**をください。／きれいな**の**がいいです。

> [!warning] のだ文との混同に注意
> 「きれいなの。」は説明の の（→ セクション3）にも読める。代用の の であることを示すには、述語をつけた文（きれいな**の**をください）で提示する。

### 2b. 名詞化の「の」— nominalização

**動作・出来事・命題そのもの**をコト化する。「〜すること」に近い。

| 例 | 意味 | PT-PT |
|---|---|---|
| 食べる**の**が好き | 食べること | Gosto **de comer** |
| 走る**の**が速い | 走ること | Corre depressa (o **correr** dele é rápido) |
| 日本へ行く**の**をやめた | 行くこと | Desisti **de ir** ao Japão |

- 置換テスト：食べるの → 食べる**こと**（○）／食べる**もの**（△ 意味が変わる）。
- PT-PT の対応：不定詞 **comer**、または名詞節 **o facto de comer**。

> [!important] の と こと の使い分け
> 知覚動詞（見る・聞く・感じる）や「待つ・手伝う・止める」などの具体的・同時的な事態は **の が必須**。
> 子供が歌う**の**を聞いた（○）／歌う**こと**を聞いた（×）
> 一方、抽象的・報告的な内容（伝える・約束する等）は **こと** を好む。

---

## 3. のだ／んです — 説明のモダリティ（の explicativo）

文末で `〜のだ／〜んです／〜の` の形をとり、理由・説明・発見・強調などを表す。名詞句ではなくモダリティ。

| 場面 | 例 | PT-PT ニュアンス |
|---|---|---|
| 質問（説明を求める） | どうした**の**？ | O que se passa? (*é que*...) |
| 説明・理由 | 疲れている**の**（です）。 | É que estou cansado. |
| 発見・納得 | 雨が降っていた**んだ**。 | Ah, **é que** estava a chover. |

- PT-PT では **é que / é porque** の説明構文がニュアンス的に近い。
- 上昇イントネーションの「〜の？」は女性語・親密体の質問にもなる。

---

## 4. 一覧対照表

| 機能 | 例 | 日本語での言い換え | PT-PT |
|---|---|---|---|
| ① 連体修飾（de） | 私の本 | — | o meu livro |
| ① 主格の代用 | 私の作った | 私が作った | que eu fiz |
| ②a 代用の の | 安いの | 安いもの | o mais barato |
| ②b 名詞化の の | 食べるのが好き | 食べること | gostar de comer |
| ③ のだ文 | どうしたの？ | — | é que...? |

---

## 5. 指導上のポイント

- ①と②は別カテゴリー（修飾 vs 準体）。②a と ②b は**同じ準体助詞だが指示対象が異なる**（もの／こと）。
- 「いずれも nominalização か？」への正確な答え：**統語的には同じ準体助詞。意味機能では ②a＝代用（pronominalização）、②b＝名詞化（nominalização）と区別する**のが正しい。
- PT-PT 対訳の型：②a は「定冠詞＋形容詞（o barato）」、②b は「不定詞（comer）」でパターン化して教えると定着しやすい。
