# 7月末決算・小法人　年間 申告・提出期限一覧

> [!info] 前提条件
> - 決算期：**7月31日**（事業年度 8月1日〜翌年7月31日）
> - 源泉所得税：**納期の特例**の承認あり（年2回納付／給与等の支給人員が常時10人未満）
> - 住民税の特別徴収：**納期の特例**の承認あり（年2回納付）
> - 消費税の**課税事業者**を前提（免税事業者は消費税関連を除く）
> - 社会保険料（健康保険・厚生年金）の納付期限は本書の対象外

## 年間スケジュール（事業年度の流れ順）

| 期限 | 項目 | 内容 | 提出・納付先 |
|---|---|---|---|
| **9月30日** ⭐ | 法人税・地方法人税・法人住民税・法人事業税の確定申告・納付 | 事業年度（8/1〜翌7/31）の確定申告。決算日の翌日から2か月以内 | 税務署／都道府県／市区町村 |
| **9月30日** ⭐ | 消費税・地方消費税の確定申告・納付 | 課税事業者の場合。決算日の翌日から2か月以内 | 税務署 |
| **12月10日** | 住民税 特別徴収（納期の特例） | 6〜11月分の特別徴収住民税をまとめて納付 | 各市区町村 |
| 年内最終給与支給日 | 年末調整の実施 | その年分の給与について過不足税額を精算 | （社内処理） |
| **1月20日** | 源泉所得税（納期の特例） | 7〜12月に源泉徴収した所得税・復興特別所得税を納付 | 税務署 |
| **1月31日** | 法定調書合計表・支払調書等の提出 | 給与所得の源泉徴収票、報酬等の支払調書、法定調書合計表 | 税務署 |
| **1月31日** | 給与支払報告書の提出 | 前年分の給与について従業員の住所地市区町村へ提出 | 各市区町村 |
| **1月31日** | 償却資産申告書の提出 | その年1月1日時点で所有する償却資産を申告 | 各市区町村 |
| **3月31日** | 法人税等の中間（予定）申告・納付 | 前期の確定法人税額が20万円超の場合。開始後6か月経過日の翌日から2か月以内 | 税務署／都道府県／市区町村 |
| **3月31日** | 消費税の中間申告・納付 | 前期の確定消費税額（国税）に応じて要否・回数が決まる（年1回の場合の期限） | 税務署 |
| **6月10日** | 住民税 特別徴収（納期の特例） | 12〜5月分の特別徴収住民税をまとめて納付 | 各市区町村 |
| 6月1日〜**7月10日** | 労働保険の年度更新 | 前年度の確定保険料の精算と当年度の概算保険料の申告・納付 | 労働局／監督署／金融機関 |
| 7月1日〜**7月10日** | 算定基礎届の提出 | 健康保険・厚生年金の標準報酬月額の定時決定（4〜6月報酬） | 日本年金機構／協会けんぽ |
| **7月10日** ⭐ | 源泉所得税（納期の特例） | 1〜6月に源泉徴収した所得税・復興特別所得税を納付 | 税務署 |

> [!note] ⭐ は決算・源泉所得税の主要期限。期限が土日祝にあたる場合は翌開庁日に繰り下げ。

## 補足・留意点

- **申告期限の延長の特例**：適用を受けている場合、確定申告期限が原則1か月延長（法人税→10月31日、消費税も1か月延長可）。
- **法人税の中間申告**：前期の確定法人税額が20万円を超える場合に必要。20万円以下なら不要。
- **消費税の中間申告**：前期の確定消費税額（国税分）で決まる。
	- 48万円以下 → 不要
	- 48万円超 400万円以下 → 年1回（期限 3月31日）
	- 400万円超 4,800万円以下 → 年3回
	- 4,800万円超 → 年11回
- **1月31日に集中する提出物**：償却資産申告・法定調書・給与支払報告書（暦年ベースで毎年固定）。
- **賞与支払届**：賞与支給日から5日以内に提出（不定期）。
- **資格取得・喪失届**：従業員の入退社時に都度提出（定例期限とは別）。

> [!warning] 本資料は一般的な期限をまとめたものです。個別の適用要件・延長特例の有無は、顧問税理士・所轄官庁でご確認ください。
