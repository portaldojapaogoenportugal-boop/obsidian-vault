# ポルトガルIRS：年金課税と控除メモ

ポルトガル居住の日本人クライアント向け。日葡租税条約による年金課税権の判定と、IRSの主要な税額控除（deduções à coleta）の上限・実務をまとめる。

---

## 1. 年金の課税権判定（日葡租税条約）

### 条約の建付け
条約（2011年署名）に**社会保障年金の特別条項はない**。年金は第17条と第18条2項のみ。

- **第17条（退職年金）**：「退職年金その他これに類する報酬は、居住者である締約国においてのみ課税」→ **居住国（ポルトガル）課税**
- **第18条2項（政府職員）**：国・地方政府等への役務に基づき政府が拠出/設立した基金から支払う退職年金 → **支払国（日本）課税**（受給者が他方の国の居住者かつ国民なら居住国課税）

### 各年金の当てはめ（民間勤務・公務員共済なし前提）

| 年金の種類 | 日本の区分 | 条約 | 課税国 |
|---|---|---|---|
| 基礎年金・厚生年金 | 公的年金（社会保険） | 第17条 | ポルトガル |
| 企業型DC（年金受給） | 私的年金（企業年金） | 第17条 | ポルトガル |
| 企業年金DB（年金受給） | 私的年金（企業年金） | 第17条 | ポルトガル |
| 公務員共済年金・恩給 | 公的年金（公務） | 第18条2項 | 日本 |

→ 公務員共済を除き、すべて第17条で**ポルトガル課税**。日本側源泉は「租税条約に関する届出書」を年金支払者経由で提出して免除。

### 用語の整理
- 厳密には**公的年金は基礎年金・厚生年金のみ**。DC・企業年金は私的年金（個人年金でもない）。
- ただし日本税法上は①②③とも「**公的年金等**」（公的年金等控除の対象）。これは条約区分とは別物。

## 2. 「natureza pública / privada」質問への回答

Finanças から `é de natureza pública (paga pelo Estado) ou privada (Fundo de Pensões)?` と聞かれた場合。

### 落とし穴
ポルトガルの「**pensão de natureza pública**」は直訳の「国が払う年金」ではなく**公務員年金（função pública＝CGA）**を指す。

- pública（公務員年金）＝ OECD第19条 ＝ 日葡条約第18条2項 → 日本課税
- privada/一般年金 ＝ OECD第18条 ＝ 日葡条約第17条 → ポルトガル課税

### 結論
厚生年金・基礎年金は「日本年金機構（国）が払う」が**función pública ではなく社会保険（Segurança Social相当）**。

→ **3種類とも「privada」**。判断は funding主体（国か基金か）ではなく**公務役務に基づくか否か**。
→ 「国が払うからpública」と誤答すると第18条2項を誤適用し、ポルトガルで誤って免除→申告漏れになる。

### 申告
外国年金は **Anexo J 第5欄**。年金コードは **H01（一般 pensões）**。H02（pensões públicas）は公務員共済のみ。

## 3. 家庭一般費（despesas gerais familiares）— CIRS 第78.º-B条

### 基本
一般支出（水道・電気・通信・スーパー・燃料・衣料等）の **35%控除**。

- 上限：各納税者（sujeito passivo）ごとに **€250**
- 合算申告：**€500**（＝€250×2）
- ひとり親：45%・€335

### 各人€250は独立（プール不可）
- 条文：第78.º-B条1項「limite global de €250 para cada sujeito passivo」、5項「各fatura記載の取得者（adquirente）ごとにATが算定」
- 計算式：`控除 = min(35%×Aの支出, €250) + min(35%×Bの支出, €250)`
- **片方の超過分を他方の不足に回せない**。満額€500には**各人約€714以上**の支出が必要（€714×35%≒€250）。
- 例：1人のNIFに€1,000集中 → 35%=€350 だが€250で頭打ち（超過切り捨て）。
- 対策：fatura発行時の **NIFを夫婦で分散**。

### 例外：扶養者支出による補填
- **両親の自己支出だけでは€500に届かない場合に限り**、子（扶養者）のNIF支出が **50/50按分**で各親に加算され、不足側を€250まで押し上げられる。
- ただし配偶者間の超過移転ではない。既に€250に達した側に乗る50%は無駄。
- Doutor Finanças「**É raríssimo**（極めて稀）」。通常は自己支出だけで€500に達するため。

#### 補填の計算例
| | 自己支出 | +子50%(€1,000の半分) | ×35% | 控除 |
|---|---|---|---|---|
| 夫A | €2,000 | +€500 | €875 | €250（頭打ち・€500分は無駄） |
| 妻B | €300 | +€500 | €280 | €250（不足を補填） |
| | | | **合計** | **€500** |

## 4. CIRS 第78.º-B条 第5項（原文と訳）

> O valor das deduções à coleta é apurado pela Autoridade Tributária e Aduaneira com base nas faturas que lhe forem comunicadas, por via eletrónica, até ao final do mês de fevereiro do ano seguinte ao da sua emissão, relativamente a cada adquirente nelas identificado.

**訳**：税額控除の額は、当該請求書（fatura）に記載された各取得者（adquirente）ごとに、その発行年の翌年2月末までに電子的方法でATに通知された請求書に基づいて、ATが算定する。

→ 控除は自己申告ではなく、**翌年2月末までにe-faturaに反映された請求書をもとに、NIF（取得者）単位でATが計算**する。

## 5. 控除総額の全体上限（CIRS 第78.º条7項）

### 対象範囲（限定列挙：c〜h、k、m号）
| 対象 | 対象外 |
|---|---|
| c) 医療・健康保険 | a) 扶養親族 |
| d) 教育・研修 | **b) 家庭一般費** |
| e) 不動産 | i) 障害者 |
| f) 扶養手当 | j) 国際二重課税 |
| g) exigência de fatura（交通費含む） | l) AIMI |
| h) 介護施設 | |
| k) 税制優遇 | |
| m) 家事使用人 | |

→ **家庭一般費（b号）は全体上限の対象外**で独立。€250/人・€500は全体上限に食われない。

### 金額（2025年所得）
- 課税所得 ≤ 第1税率（≈€8,059）→ **上限なし**
- €8,059超〜€80,000以下 → **算式**：`€1,000 + €1,500×(€80,000−RC)/(€80,000−€8,059)`（範囲€1,000〜€2,500）
- €80,000超 → **€1,000**
- 合算申告：第69.º条除数（÷2）を課税所得に適用してから区分判定（有利）
- 扶養3人以上：1人につき上限+5%（第78.º条8項）

## 6. 健康保険料の控除（CIRS 第78.º-C条1項c）

- 健康リスクのみ担保する健康保険料は**医療費として15%・上限€1,000**で控除可。
- 経路：保険会社 → 毎年1月 **Modelo 37** でATへ申告 → **Anexo H**（保険料欄、年度により6-C/6-C1）へ自動プリフィル。**e-fatura経由ではない**。
- e-faturaでは保険会社が複数CAEを持つため pendentes に落ち「**Outros**」しか選べないことがあるが、保険料控除に影響しない。saúdeに入れようとする必要も放置の問題もない。
- 確認は e-fatura ではなく **Modelo 3 の Anexo H** を開き、保険料行のプリフィルを直接チェック。
- 出てこない場合：①保険会社がModelo 37未提出/誤り → Anexo H手入力で訂正可、②死亡保障等の健康外カバー混在で対象外。
- 種類：健康保険＝控除可／生命・住宅・自動車・賠償＝対象外（混合契約は健康部分が区分できなければ全額不可）。

### 移住エージェント費用
- 専用カテゴリーなし → 家庭一般費（b号）どまり。上限€250/人で通常は他の支出で吸収。
- そもそもポルトガルNIF付き正式fatura＋e-fatura通知がなければ計上不可（移住前・海外業者は対象外）。

## 7. 交通費控除（CIRS 第78.º-D条）

### 対象
**公共集団旅客輸送の月間定期券（passes mensais）または乗車券（bilhetes）**で、発行事業者が以下のCAE（セクションH）に該当するもの。

| CAE | 内容 | 例 |
|---|---|---|
| 49310 | 都市・近郊の陸上旅客輸送 | 地下鉄、市内バス、市電、近郊路線（Metro de Lisboa, Carris, STCP） |
| 49391 | 都市間バス輸送 | 長距離・都市間バス（Rede Expressos等） |
| 49392 | その他陸上旅客輸送 | ケーブルカー、フニクラ等 |
| 50102 | 沿岸・域内の水上旅客輸送 | フェリー（Transtejo/Soflusa等） |
| 50300 | 内陸水路の旅客輸送 | 河川輸送 |

### 控除の建付け
- **IVAの100%**を控除（運賃総額ではない）。公共交通は軽減税率6%なので、例：月€40の定期券→IVA分 約€2.26（€40×6/106）が対象。
- 独立枠ではなく「**exigência de fatura**」の世帯**€250**上限（第78.º-D条1項）に合算。
- fatura にNIF記載＋e-fatura通知が要件。
- **全体上限（第78.º条7項 g号）の対象**に含まれる点に注意（家庭一般費とは異なる）。
