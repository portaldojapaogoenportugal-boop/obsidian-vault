# 日葡税務 根拠条文まとめ

## 日葡租税条約（1969年署名・1970年発効）

| 条文 | 内容 |
|------|------|
| 第6条 | 不動産所得 |
| 第17条 | 年金課税 → **ポルトガルのみ課税** |
| 第18条第2項 | 公務員年金のみ適用（支払国課税）。国民年金には**適用されない** |
| 第23条 | 二重課税の排除 |

### 年金の取り扱い（第17条）
- 国民年金・企業年金・確定拠出年金 → すべて第17条適用
- 第18条第2項（支払国課税）は「役務提供の対価」である公務員年金に適用される規定
- 国民年金は社会保険制度のため第17条が適用される
- 日本で源泉徴収されていないのは条約上**正しい取り扱い**

## CIRS（ポルトガル所得税法）

| 条文 | 内容 |
|------|------|
| Art. 13º・14º | 夫婦合算（Tributação conjunta）vs 個別申告 |
| Art. 15º | 世界所得課税原則（居住者は全世界所得を申告） |
| Art. 16º | 居住者認定（183日ルール） |
| Art. 53º | 年金所得控除：**8.54 × IAS**（Lei n.º 45-A/2024 改正。旧：4,104€ 固定） |
| Art. 78º-A | 高齢者控除：65歳以上 **+1,721€** |
| Art. 78º-E | 家賃控除：支払家賃の **15%・上限700€**（Lei n.º 36/2024）※Finançasへの賃貸契約登録が条件 |

## NHR・IFICI

| 制度 | 適用可否 | 理由 |
|------|----------|------|
| 旧NHR | **対象外** | 2025年移住のため |
| 経過措置NHR | **対象外** | 2024年12月31日までの移住が条件 |
| IFICI（新NHR） | **原則対象外** | D7・年金生活者は職種要件を満たさない |

> 根拠：EBF Art. 58-Aº、Portaria n.º 352/2024

## RGIT（罰則）

- NIF居住者変更届出違反：**75€〜375€**（夫婦各々）
- 遡及適用（移住日からの届出）の手続きが存在する

## IRS申告（2025年分）

- **申告期限：2026年6月30日**
- NIFの居住者変更が完了していないと申告自体ができない
- 賃貸収入（世界所得課税原則）：日本での税額ゼロのため外国税額控除なし
