### 🧩 Ponto de gramática – 文法ポイント

#### Estarの用法

この課では、estar動詞の用法が2種類出てきました。一つは、**estar incluído****（**含まれている）、もう一つは**estar com fome**（お腹が空いた）です。それぞれ、以下説明します。

##### Estar＋形容詞

ホテルに宿泊する場合、宿泊代に朝食が含まれているかどうかの確認は頻出です。ここで、estáは第3課で説明した通り、estar動詞の三人称単数形です。非恒常的状態を表すために、形容詞と組み合わせて使われます。ここでは、incluído（含まれる）という形容詞と組み合わせて使われています。逆に、受付で朝食が含まれているか尋ねる場合は、全く同じ文章を文末のイントネーションを上げて発話すれば、疑問文になります。

**O pequeno-almoço está incluído?**

ここでは、朝食pequeno-almoçoが男性名詞のため形容詞も男性形ですが、もし主語が女性形なら、形容詞も女性形になり、例えば以下のようになります。また、主語が複数形場合、形容詞も複数形になります。

| **ポルトガル語**                    | **日本語訳**       | **英語訳**              | **備考** |
| ------------------------------ | -------------- | -------------------- | ------ |
| **A água está fria.**          | 水は冷たいです。       | The water is cold.   | (1)    |
| **Os quartos estão limpos.**   | 部屋はきれいです。      | The rooms are clean. | (2)    |
| **As meninas estão cansadas.** | 女の子たちは疲れています。 | The girls are tired. | (3)    |

(1)   水を意味するáguaは女性名詞なので、冷たいを意味する形容詞は女性形friaとなります。

(2)   部屋を意味するquartoは男性名詞で、その複数形quartosが主語なので、きれいを意味する形容詞limpoも男性形かつ複数形limposとなります。

(3)   女の子を意味するmeninaは女性名詞で、その複数形meninasが主語なので、疲れているを意味する形容詞cansadoも女性形かつ複数形cansadasとなります。

##### Estar com＋名詞

形容詞の場合は、その形容詞をestarにつければ、その非恒久的状態を表すことができますが、名詞を使った表現方法もあります。その場合、前置詞comをつけて表現することになります。ここの文脈では、comは英語の前置詞withのようなもの、と考えれば、分かりやすいかと思います。この構文を使った表現として、例えば、以下のような例があります。例文は全て主語が一人称単数（私）ですが、主語が変われば、estarも変化します。

| **ポルトガル語**                       | **日本語訳**     | **英語訳**               | **備考** |
| --------------------------------- | ------------ | --------------------- | ------ |
| **Estou com sede.**                | のどが渇いています。  | I'm thirsty.          |        |
| **Estou com sono.**                | 眠いです。       | I'm sleepy.           |        |
| **Estou com frio.**                | 寒いです。       | I'm cold.             | (1)    |
| **Estou com calor.**               | 暑いです。       | I'm hot.              |        |
| **Estou com febre.**               | 熱があります。     | I have a fever.       |        |
| **Estou com dores de cabeça.**     | 頭が痛いです。     | I have a headache.    | (2)    |

(1)   frioは名詞・形容詞両方の意味を持ち、ここでは名詞として使われています。

(2)   doresが痛みを意味します。頭を意味するcabeçaのほか、胃（estômago）や歯（dentes）など、その他体の部位をcabeçaと入れ替えれば、その場所の痛みを訴えることができます。

尚、このestar com部分は第3章第1課で学習したterに置き換えて使うこともできます。例えば、お腹が空いた、ならTenho fomeとなります。

### 💬 Dica cultural – 文化メモ

朝ごはんは、ビュッフェスタイルのほか、コンチネンタルスタイル（continental）もよく見かけます。ポルトガルの朝ごはんでパンにつけるものと言えば、一般的なジャムのほか、ポルトガルの典型的なものとしてマルメラーダ（marmelada）があります。マルメロ（marmelo）という果実を砂糖で煮込んで作られ、ジャムよりは硬めの固形で、切って食べます。パンにのせて食べてもいいし、食後のデザートとしてチーズと一緒に食べられることもよくある光景です。各家庭で作って保存されていることも多く、その場合は市販のものより糖分が控えめだったりなど、それぞれの家庭の母の味が味わえます。

そのほか、知っていたら便利な朝食ワードを挙げておきます。
