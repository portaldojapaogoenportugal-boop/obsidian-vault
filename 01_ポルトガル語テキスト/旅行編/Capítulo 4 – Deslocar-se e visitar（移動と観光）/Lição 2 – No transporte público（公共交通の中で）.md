### 🗣️ Diálogo – 会話例

Revisor: Bom dia. O seu bilhete, por favor.

Viajante: Aqui está.

Revisor: Desculpe, este lugar é o 14B. O seu é o 16A.

Viajante: Ah, sim? Desculpe! Onde é o 16A?

Revisor: É ali, junto à janela.

Viajante: Obrigada. O comboio pára em Coimbra?

Revisor: Sim, pára. É a próxima paragem.

Viajante: Está bem, obrigada.

Revisor: De nada.

### 💬 Tradução – 日本語訳

車掌：おはようございます。切符をお願いします。

旅行者：どうぞ。

車掌：すみませんが、この席は14Bです。お客様の席は16Aです。

旅行者：あ、そうですか？すみません！16Aはどこですか？

車掌：あちらです。窓のそばです。

旅行者：ありがとうございます。この電車はコインブラに止まりますか？

車掌：はい、止まります。次の停車駅です。

旅行者：わかりました。ありがとうございます。

車掌：どういたしまして。

### 🇬🇧 English translation – 英語訳

Conductor: Good morning. Your ticket, please.

Traveller: Here you are.

Conductor: Excuse me, this seat is 14B. Yours is 16A.

Traveller: Oh, really? Sorry! Where is 16A?

Conductor: It’s there, next to the window.

Traveller: Thank you. Does the train stop in Coimbra?

Conductor: Yes, it does. It’s the next stop.

Traveller: All right, thank you.

Conductor: You’re welcome.

### 🧩 Ponto de gramática – 文法ポイント

1️⃣ 前置詞 em / para / de

|   |   |   |
|---|---|---|
|前置詞|意味|例文|
|em|〜の中・〜で|Estou em Lisboa.（リスボンにいます）|
|para|〜へ（方向）|O comboio vai para o Porto.（電車はポルトへ行きます）|
|de|〜から（出発）|O comboio vem de Lisboa.（電車はリスボンから来ます）|

2️⃣ 動詞 parar（止まる）

|   |   |   |
|---|---|---|
|Pessoa|Forma|Exemplo|
|Eu|paro|Eu paro aqui.（私はここで止まります）|
|Ele / O comboio|pára|O comboio pára em Coimbra.（電車はコインブラに止まります）|

💡 ポルトガルでは「止まる」＝ pára（アクセント付き）と書くのが正しい表記。

3️⃣ 副詞 aqui / ali / lá

|   |   |   |
|---|---|---|
|Português|日本語|説明|
|aqui|ここ|話し手の近く|
|ali|そこ|聞き手や中間地点|
|lá|あそこ|離れた場所|

### 💬 Dica cultural – 文化メモ（Portugal）

・ポルトガルでは、座席指定付きの列車（Alfa Pendular, Intercidades）では座席番号が指定されている。

・車掌（revisor）は切符をチェックし、座席を案内してくれることもある。

・窓側＝junto à janela、通路側＝junto ao corredor。

・「次の停車駅」＝a próxima paragem は旅行中によく使う表現。
